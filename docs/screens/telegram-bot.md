# 텔레그램 봇

## 목적
서비스 핵심 허브. 온보딩 + 설정 + 리포트 수신.

## 봇 정보
- 봇 이름: StockReadBot (예정)
- 라이브러리: python-telegram-bot

## 명령어
| 명령 | 동작 |
|------|------|
| /start | 온보딩 |
| /my | 현재 설정 확인 |
| /edit | 성향 + 종목 수정 (온보딩 버튼 뷰 재표시) |
| /help | 명령어 안내 |

## 온보딩 플로우
1. /start → 환영 메시지
2. 투자 성향 선택 (인라인 버튼: 안정/중립/공격)
3. 종목 선택 (인라인 버튼: 미국 6개 + 한국 6개, 최대 3개)
4. 설정 완료 확인

## 리포트 발송
- 시간: 매일 07:00 KST
- 방식: 종목별 개별 메시지 (3종목 = 3개 메시지)
- 면책 문구 포함

## 엣지 케이스
| 상황 | 처리 |
|------|------|
| 종목 3개 초과 | "최대 3개까지 가능. 다른 종목 해제 먼저" |
| 이미 구독 중 종목 | 토글 해제 |
| 온보딩 미완료 시 /my | 온보딩 유도 |
