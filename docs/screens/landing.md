# 웹 랜딩 페이지

## 목적
서비스 소개 + 텔레그램 봇 연결 유도

## 구성 요소
- [ ] 히어로: 후킹 카피 + CTA 버튼. 임팩트 비주얼
- [ ] 리포트 예시: 종목 탭으로 골라서 볼 수 있음 (인터랙티브)
- [ ] 특징 3가지: ① AI 해석 ② 성향 맞춤 ③ 매일 아침 자동
- [ ] 하단 CTA: "무료로 시작하기" → t.me/StockReadBot

## 디자인 방향
- 프리미엄 + 깔끔
- 후킹 되는 예쁜 UI
- 모바일 퍼스트

## 데이터
- 리포트 예시 데이터: stocks 테이블 + daily_reports 최신

## 액션
- CTA 클릭 → 텔레그램 봇 열기 (t.me/StockReadBot)
- 종목 탭 클릭 → 해당 종목 리포트 예시 표시

## 기술
- Next.js 정적 페이지
- 반응형 (모바일 우선)
- Vercel 배포
