## 섹션 구성

### 1. 네비게이션
- 좌: 로고 아이콘 (28px, 그린 그라데이션) + "주읽이" 텍스트
- 우: "시작하기" CTA 버튼 (그린, 8px radius)

### 2. 히어로
- 라이브 펄스 뱃지: "매일 아침 7시 리포트 발송 중" (그린 glow)
- 헤드라인: "주식 뉴스, 읽어드립니다" (32px, 800w, 그린 강조)
- 서브카피: "AI가 매일 아침 당신의 종목을 초보 눈높이로 해석해 드려요."
- CTA: "텔레그램에서 무료로 시작 →" (52px height, 12px radius, 그린)
- 하단 텍스트: "가입 없이 바로 시작 · 완전 무료"

### 3. 리포트 프리뷰 (인터랙티브)
- 종목 탭: NVDA / TSLA / 삼성전자 / AAPL / SK하이닉스 (pill 스타일)
- 성향 버튼: 🛡 안정형 (파랑) / ⚖️ 중립형 (보라) / 🔥 공격형 (빨강)
- 리포트 카드:
  - 헤더: 국기 + 티커 + 이름 + 가격 + 등락률 + 뱃지 2개
  - 본문 4섹션: 지금 어디쯤이에요? / 차트가 말해주는 것 / 무슨 일이 있었나요? / 이렇게 보시면 돼요
  - 해석 섹션: 녹색 배경 카드로 강조 (성향별 톤 변경)
  - 푸터: 면책 문구

### 4. 특징 3가지
- 카드 형태 (16px radius, #111113 배경)
- 🧠 데이터가 아닌 해석 (파란 아이콘 배경)
- 🎯 투자 성향 맞춤 (보라 아이콘 배경)
- ⏰ 매일 아침 자동 배달 (그린 아이콘 배경)

### 5. How it works
- 3단계 세로 스텝 (그린 원형 넘버 + 연결선)
- 1: 텔레그램 봇 시작 → 2: 성향 & 종목 선택 → 3: 매일 아침 리포트 수신

### 6. 하단 CTA
- "내일 아침, 첫 리포트를 받아보세요" (그린 강조)
- "무료로 시작하기 →" 버튼

### 7. 푸터
- 로고 + © 2026 StockRead
- 면책: "이 서비스는 참고용이에요. 투자 판단은 본인이 직접 해주세요!"
