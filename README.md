# 주읽이 (StockRead) 📊

> 주린이를 위한 AI 기반 투자 해석 서비스

매일 아침 7시, AI가 당신의 종목을 초보 눈높이로 해석해 텔레그램으로 보내드려요.

## 서비스 채널
- **Web** — 랜딩 페이지 (소개 + 텔레그램 유도)
- **Telegram Bot** — 온보딩 + 일일 리포트 수신
- **YouTube Shorts** — 매일 시황 요약 영상 (자동 생성)

## 기술 스택
| 영역 | 기술 |
|------|------|
| Web | Next.js + TypeScript (Vercel) |
| Backend | Python + FastAPI |
| DB | Supabase (PostgreSQL) |
| AI | Claude API |
| TTS | OpenAI TTS |
| 영상 | Remotion |
| 봇 | python-telegram-bot |

## 프로젝트 구조
```
├── docs/       기획 + 디자인 스펙
├── web/        Next.js 랜딩 페이지
├── backend/    Python FastAPI 서버
└── remotion/   쇼츠 영상 렌더링
```
